## 安装
1. 克隆 [admin](https://github.com/ydtg1993/admin.git)
2. 命令 composer update
3. 在项目根目录执行 php artisan key:generate
3. 创建mysql数据库 导入项目database目录的admin.sql 

   `默认账号: admin 密码: 123456`
